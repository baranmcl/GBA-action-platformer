# Spronk Quest — Milestone 1 (Vertical Slice) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a playable GBA vertical slice where Laurel runs, jumps, and fires a magic bolt through a short hub + Dungeon 1, frees the caged spronk to earn Featherleap (double-jump), which opens a gated exit — with health/magic meters, a title screen, and an SRAM save — booting at 60fps in mGBA and on real hardware.

**Architecture:** Three-layer split. `src/logic/` is pure C++17 with **zero Butano types** (`bn::` forbidden) and is exercised by host g++ unit tests — it owns fixed-point math, physics, collision, player/meter/world state. `src/engine/` is the Butano glue (input adapter, camera, sprite/anim, HUD, SRAM). `src/game/` holds the scene state machine and Dungeon-1 content. Level/dungeon layouts are authored as data and compiled to C++ headers by a Python tool (the proven `gameboygame` pattern). Placeholder art is generated by a Python/Pillow script and is a drop-in PNG replacement later.

**Tech Stack:** devkitARM, Butano (C++17 GBA engine), Python 3 + Pillow (asset/level codegen), g++ (host unit tests), mGBA (emulator), GNU Make.

**BEFORE marking any task complete:**
1. Review tests against `docs/pitfalls/testing-pitfalls.md` (error paths? edge cases?).
2. Run the relevant test command and confirm green.
3. Commit with a message that states what happened to test assertions ("add"/"strengthen"/"preserve"; never an obscuring "CI fix").

**Assertion rigor under pressure (timing/flaky tests):** If any assertion races, flakes, or fails nondeterministically, the fix is deterministic synchronization, NOT assertion removal or weakening. If synchronization cannot make it pass reliably, STOP and raise to the dispatching agent. Prefer mechanism assertions over symptom assertions.

**Logic-vs-glue test convention:**
- `src/logic/**` tasks are **host-unit-tested** with g++ via `make -C test` — full TDD with real assertions.
- `src/engine/**` and `src/game/**` tasks that touch Butano/hardware **cannot** be host-unit-tested. Their "test" is a scripted **mGBA verification**: build the ROM, run it, and confirm the explicit observable acceptance listed in the task. Where logic can be extracted from glue, extract it into `src/logic/` and unit-test it there instead. **No `bn::` type may appear under `src/logic/`** — Task 1.0 adds a guard that fails the build if it does.

**After completing each Phase:** Review the batch from multiple perspectives. Minimum 3 review rounds; if round 3 still finds issues, keep going until clean. (Phase 1 additionally runs `/plan-review-cycle`-style scrutiny on the logic API since all later phases depend on it.)

#include <cstdio>
#include <vector>
#include <functional>

struct TestCase { std::string name; std::function<void()> fn; };
inline std::vector<TestCase>& registry() { static std::vector<TestCase> r; return r; }
inline int& failures() { static int f = 0; return f; }
struct Registrar { Registrar(const char* n, std::function<void()> f){ registry().push_back({n,f}); } };
#define TEST(name) static void name(); static Registrar reg_##name(#name, name); static void name()
#define CHECK(cond) do { if(!(cond)){ ++failures(); std::printf("  FAIL %s:%d  CHECK(%s)\n", __FILE__, __LINE__, #cond);} } while(0)
#define CHECK_EQ(a,b) do { auto _a=(a); auto _b=(b); if(!(_a==_b)){ ++failures(); std::printf("  FAIL %s:%d  CHECK_EQ(%s,%s)  got %lld vs %lld\n", __FILE__, __LINE__, #a,#b,(long long)_a,(long long)_b);} } while(0)

```cpp
#pragma once
#include <cstdint>
namespace logic {
struct Fixed {
    int32_t raw; // 16.8 fixed point
    static constexpr int SHIFT = 8;
    static constexpr Fixed from_raw(int32_t r){ return Fixed{r}; }
    static constexpr Fixed from_int(int32_t i){ return Fixed{ i << SHIFT }; }
    constexpr int32_t to_int() const { return raw >> SHIFT; } // arithmetic shift floors toward -inf
    constexpr Fixed operator+(Fixed o) const { return {raw + o.raw}; }
    constexpr Fixed operator-(Fixed o) const { return {raw - o.raw}; }
    constexpr Fixed operator-() const { return {-raw}; }
    constexpr Fixed operator*(Fixed o) const { return { (int32_t)(((int64_t)raw * o.raw) >> SHIFT) }; }
    constexpr bool operator==(Fixed o) const { return raw == o.raw; }
    constexpr bool operator<(Fixed o) const { return raw < o.raw; }
    constexpr bool operator>(Fixed o) const { return raw > o.raw; }
    constexpr bool operator<=(Fixed o) const { return raw <= o.raw; }
    constexpr bool operator>=(Fixed o) const { return raw >= o.raw; }
};
inline constexpr Fixed operator""_fx(unsigned long long i){ return Fixed::from_int((int32_t)i); }
}
```

#pragma once
#include <cstdint>
#include "logic/fixed.h"
namespace logic {
enum class TileKind : uint8_t { Empty=0, Solid=1, OneWay=2 };
struct Tilemap {
    int w, h;
    const uint8_t* cells; // row-major, length w*h
    static constexpr int TILE = 8;
    TileKind at(int tx, int ty) const {
        if(tx<0 || ty<0 || tx>=w || ty>=h) return TileKind::Solid; // OOB = wall
        return static_cast<TileKind>(cells[ty*w + tx]);
    }
    bool is_solid(int tx,int ty) const { return at(tx,ty)==TileKind::Solid; }
    bool is_oneway(int tx,int ty) const { return at(tx,ty)==TileKind::OneWay; }
    static int px_to_tile(Fixed px){ return px.to_int() / TILE; }
};

// src/logic/collision.cpp
#include "logic/collision.h"
namespace logic {
static bool overlaps_solid(const Body& b, const Tilemap& map){
    int l = Tilemap::px_to_tile(b.pos.x);
    int r = Tilemap::px_to_tile(b.pos.x + b.half_w + b.half_w - Fixed::from_int(1));
    int t = Tilemap::px_to_tile(b.pos.y);
    int bm= Tilemap::px_to_tile(b.pos.y + b.half_h + b.half_h - Fixed::from_int(1));
    for(int ty=t;ty<=bm;++ty) for(int tx=l;tx<=r;++tx) if(map.is_solid(tx,ty)) return true;
    return false;
}
void move_and_collide(Body& b, const Tilemap& map){
    b.on_ground = false;
    const Fixed step = Fixed::from_int(Tilemap::TILE); // ≤ one tile per substep -> no tunneling
    // ---- X axis ----
    Fixed remx = b.vel.x;
    while(remx.raw != 0){
        Fixed d = remx;
        if(d > step) d = step; if(d < -step) d = -step;
        b.pos.x = b.pos.x + d;
        if(overlaps_solid(b,map)){
            // back out by integer pixels until clear
            Fixed dir = (d.raw>0)? Fixed::from_int(1) : Fixed::from_int(-1);
            while(overlaps_solid(b,map)) b.pos.x = b.pos.x - dir;
            b.vel.x = Fixed::from_int(0); break;
        }
        remx = remx - d;
    }
    // ---- Y axis ----
    Fixed prev_bottom = b.pos.y + b.half_h + b.half_h;
    Fixed remy = b.vel.y;
    while(remy.raw != 0){
        Fixed d = remy; if(d>step) d=step; if(d<-step) d=-step;
        Fixed before = b.pos.y;
        b.pos.y = b.pos.y + d;
        bool hit = overlaps_solid(b,map);
        // one-way: only when moving down and previous bottom was above the platform's top
        if(!hit && d.raw>0){
            int l=Tilemap::px_to_tile(b.pos.x);
            int r=Tilemap::px_to_tile(b.pos.x + b.half_w+b.half_w - Fixed::from_int(1));
            int bm=Tilemap::px_to_tile(b.pos.y + b.half_h+b.half_h - Fixed::from_int(1));
            for(int tx=l;tx<=r && !hit;++tx){
                if(map.is_oneway(tx,bm)){
                    Fixed plat_top = Fixed::from_int(bm*Tilemap::TILE);
                    if(prev_bottom <= plat_top) hit = true;
                }
            }
        }
        if(hit){
            Fixed dir = (d.raw>0)? Fixed::from_int(1):Fixed::from_int(-1);
            while(overlaps_solid(b,map)) b.pos.y = b.pos.y - dir;
            if(d.raw>0) b.on_ground = true;
            b.vel.y = Fixed::from_int(0);
            (void)before; break;
        }
        remy = remy - d;
    }
}
}

#include "logic/player.h"
namespace logic {
static const Fixed RUN_ACCEL  = Fixed::from_raw(24);
static const Fixed RUN_MAX    = Fixed::from_int(2);
static const Fixed FRICTION   = Fixed::from_raw(24);
static const Fixed JUMP_VY    = Fixed::from_int(-4);
static const PhysicsParams PH { Fixed::from_raw(28), Fixed::from_int(6) }; // gravity, terminal
void Player::update(const InputFrame& in, const Tilemap& map){
    // horizontal
    if(in.right){ body.vel.x = body.vel.x + RUN_ACCEL; facing=1; }
    else if(in.left){ body.vel.x = body.vel.x - RUN_ACCEL; facing=-1; }
    else { // friction toward 0
        if(body.vel.x.raw>0){ body.vel.x = body.vel.x - FRICTION; if(body.vel.x.raw<0) body.vel.x=Fixed::from_int(0);} 
        else if(body.vel.x.raw<0){ body.vel.x = body.vel.x + FRICTION; if(body.vel.x.raw>0) body.vel.x=Fixed::from_int(0);} }
    if(body.vel.x>RUN_MAX) body.vel.x=RUN_MAX; if(body.vel.x<-RUN_MAX) body.vel.x=-RUN_MAX;
    // jump
    if(in.jump_pressed){
        if(body.on_ground){ body.vel.y=JUMP_VY; }
        else if(abilities.featherleap && air_jumps_left>0){ body.vel.y=JUMP_VY; --air_jumps_left; }
    }
    apply_gravity(body, PH);
    bool was_ground = body.on_ground;
    move_and_collide(body, map);
    if(body.on_ground && !was_ground){ air_jumps_left = abilities.featherleap?1:0; }
    if(body.on_ground) air_jumps_left = abilities.featherleap?1:0;
}
}

```cpp
#pragma once
#include <cstdint>
#include <type_traits>
namespace logic {
struct World { bool spronk1_freed=false, featherleap=false, gate1_open=false; };
struct SaveData {
    uint32_t magic;     // 'SPRK'
    uint16_t version;   // 1
    uint16_t flags;     // bit0 spronk1, bit1 featherleap, bit2 gate1
    uint8_t  checksum;  // sum of first 8 bytes (magic,version,flags) & 0xFF; excludes itself + pad
    uint8_t  _pad[3];
};
// bn::sram::write/read require a trivially-copyable, fixed-size record. Lock both:
static_assert(std::is_trivially_copyable<SaveData>::value, "SaveData must be trivially copyable for SRAM");
static_assert(sizeof(SaveData) == 12, "SaveData layout changed — bump SAVE_VERSION and migrate");
static constexpr uint32_t SAVE_MAGIC = 0x5350524B; // "SPRK"
static constexpr uint16_t SAVE_VERSION = 1;
inline uint8_t calc_checksum(const SaveData& s){
    const uint8_t* p=reinterpret_cast<const uint8_t*>(&s); uint32_t sum=0;
    for(int i=0;i<8;++i) sum+=p[i]; // magic(4)+version(2)+flags(2)
    return (uint8_t)(sum & 0xFF);
}
inline SaveData make_save(const World& w){
    SaveData s{}; s.magic=SAVE_MAGIC; s.version=SAVE_VERSION;
    s.flags = (w.spronk1_freed?1:0)|(w.featherleap?2:0)|(w.gate1_open?4:0);
    s.checksum = calc_checksum(s); return s;
}
inline bool load_save(const SaveData& s, World& out){
    if(s.magic!=SAVE_MAGIC || s.version!=SAVE_VERSION) return false;
    if(s.checksum != calc_checksum(s)) return false;
    out.spronk1_freed = s.flags&1; out.featherleap=s.flags&2; out.gate1_open=s.flags&4;
    return true;
}
}
```

```cpp
#pragma once
#include "bn_optional.h"
namespace game {
enum class SceneId { Title, Play };
// A scene runs one frame and optionally requests a transition.
// Implemented as free functions per scene to keep state local (see scene_title/scene_play).
}
```
  `src/main.cpp` owns the scene switch and the single Butano frame pump (exactly one `bn::core::update()` per frame, at the bottom of the loop — never call it inside a scene):
```cpp
#include "bn_core.h"
#include "game/scene.h"
#include "game/scene_title.h"
#include "game/scene_play.h"
int main(){
    bn::core::init();
    game::SceneId scene = game::SceneId::Title;
    bool first = true;                         // true on the first frame after entering a scene
    while(true){
        bn::optional<game::SceneId> next;
        switch(scene){
            case game::SceneId::Title: next = game::run_title_frame(first); break;
            case game::SceneId::Play:  next = game::run_play_frame(first);  break;
        }
        first = false;
        if(next){ scene = *next; first = true; }   // re-enter the target scene fresh next iteration
        bn::core::update();   // the ONE frame advance
    }
}
```
  > Each scene exposes `bn::optional<SceneId> run_<name>_frame(bool first)`. It keeps persistent state in a file-local `static` struct, and **when `first==true` it MUST (re)initialize that state** (rebuild sprites/bg, reset player/meters/position). This makes Title→Play→Title→Play re-entry deterministic — without it, stale sprites from a prior visit leak. Mirrors the `gameboygame` one-file-per-scene pattern. Do NOT call `bn::core::update()` inside any scene (pitfall: double frame-advance). NOTE: Butano objects are RAII — on scene exit, ensure the prior scene's `static bn::optional<...>` handles are reset/destroyed before the next scene builds its own, to avoid exceeding sprite/bg limits.
